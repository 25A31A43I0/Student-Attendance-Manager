Student Attendance Manager

A simple C++ console-based project to manage student attendance records using Object-Oriented Programming (OOP) concepts and STL vector.

Features
Add student details
Store:
Roll Number
Student Name
Attendance Percentage
Display all student records
Search student by Roll Number
User-friendly menu system
